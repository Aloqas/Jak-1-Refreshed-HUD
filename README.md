# Jak 1 - Refreshed HUD
Handmade HD HUD texture pack for the OpenGOAL PC port of Jak 1.

## Setup
1. Download latest release and unzip
3. Create a new ZIP file from `custom_assets`, `metadata.json`, `cover.png` and `README.md`
4. Open the OpenGOAL Launcher
5. Go to Jak & Daxter: The Precursor Legacy -> Features -> Texture Packs
6. Click 'Add New Pack' and select the new ZIP
7. Click 'Apply Texture Changes'
8. Wait for it to finish decompiling
9. Enjoy!
